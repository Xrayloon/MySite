*** МЕСТО ДЛЯ ИНСТРУКЦИИ ПО ЗАПУСКУ У СЕБЯ НА КОМПЕ ***


Привет, это мой первый проект по созданию небольшого веб-приложения на FastAPI
Стак моего проекта: Python(3.14.3), FastAPI(0.129.0), PostgresSQL, SQLAlcemy, Alembic  

Цель: Создать небольшое Веб-приложение(Блог + интернет магазин) с небольшим знанием Питона,
Далее к нему сделат ь "чат тех.поддержки"

1)Я не использую AI для генерации кода, максиму для объяснении непонятных мне концепции и команд
2) По времени разработка должна занять не более 2-ух месяцев (парарельно защищая диплом)
